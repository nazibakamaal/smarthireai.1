# AI Powered Resume Screening System

## 📌 Problem Statement
Companies receive hundreds of resumes for a single job role.
Manual screening is time-consuming and sometimes biased.
We need an AI system to automatically analyze and rank resumes.

## 💡 Solution
- Extract text from PDF resumes
- Compare with job descriptions using TF-IDF & cosine similarity
- Rank candidates automatically
- Show matching score in percentage

## 🚀 Features
- Upload PDF resumes
- Automatic scoring & ranking
- User-friendly interface
- Hackathon-ready project
